# Android-Notepad
 Notepad
Notepad application made with Kotlin

# Technologies 
- Used MVVM architecture.
- Room
- Databinding, LiveData , ViewModel
- Navigation Component
# Features 
- View Note List
- Add Note
- Update Note
- Delete Note
- Search Note
